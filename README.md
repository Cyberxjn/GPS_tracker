A powerful GPS tracking tool for educational purposes, location monitoring, and security research. Supports real-time location tracking, geofencing, and location history.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Real-time Tracking** | Live location updates with map integration |
| **Location History** | Store and view historical location data |
| **Geofencing** | Set virtual boundaries with alerts |
| **Multiple Platforms** | Android, iOS, and Web support |
| **Export Data** | Export location history to CSV/JSON |
| **SMS Commands** | Control via SMS (Android) |
| **Web Dashboard** | Real-time web interface |
| **Telegram Bot** | Track via Telegram |

### 🚀 Key Capabilities
- 📍 **Real-time GPS coordinates** - Latitude, Longitude, Altitude
- 🗺️ **Map Integration** - OpenStreetMap / Google Maps
- 📊 **Analytics** - Speed, distance, and time tracking
- 🔔 **Instant Alerts** - Entry/exit geofence notifications
- 💾 **Offline Storage** - Local database for history
- 📱 **Cross-Platform** - Works on Android, iOS, Linux

---

## 📋 Prerequisites

- Python 3.7 or higher
- Internet connection
- GPS-enabled device
- For Android: Termux app
- For Web: Modern browser
